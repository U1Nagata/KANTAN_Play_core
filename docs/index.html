ファームウェアを書き込む

下のボタンを押して、USBケーブルで接続したM5Stack CoreS3にファームウェアを書き込みます。

お使いのブラウザは Web Serial API に対応していません。
Google Chrome または Microsoft Edge（デスクトップ版）をお使いください。

このページはHTTPS接続でアクセスする必要があります。

書き込み手順

  1. M5Stack CoreS3 をUSBケーブルでPCに接続します
  2. 上の「書き込み開始」ボタンをクリックします
  3. シリアルポート選択ダイアログが表示されたら、該当するポートを選択します
  4. 書き込みが完了するまでお待ちください（約1〜2分）
  5. 書き込み完了後、デバイスが自動的に再起動します

注意事項

  • 対応ブラウザ: Google Chrome または Microsoft Edge のデスクトップ版が必要です。 Safari、Firefox、モバイルブラウザでは動作しません。
  • KANTAN Play base: このアプリケーションは専用ハードウェア「KANTAN Play base」と組み合わせて使用するものです。 KANTAN Play base が接続されていない状態では起動しません。
  • 初回書き込み: 初めて書き込む場合は、フラッシュの全消去を行うオプションが表示されます。 通常は全消去をお勧めします。
  • USBドライバ: デバイスが認識されない場合は、USBドライバのインストールが必要な場合があります。
